Grault => Modular Cohesion

This module features service modularity and cohesion among them through messaging using RxJS among services.

  • Notify button is handled by GraultController and it calls GraultService.processNotify(), representing business activity happening in Grault module.
  • GraultService.processNotify() futher calls MessagingService.publish() which acts as broker to dispatch the message to subscribers.
  • Subscribers bring their service in execution context from run block supported by Angular modules and subscribes to MessagingService.
  • Baz, Quux and Corge modules subscribe to this message and displays it as notification at top of the page.
